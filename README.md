# Horizon Real Estate Website

## Overview
Horizon is a modern real estate website built to showcase premium properties and generate high-quality leads for real estate businesses. The project focuses on clean design, fast performance, and clear conversion paths rather than unnecessary complexity.

This is a front-end focused application designed to be easily customizable and reusable for multiple clients.

---

## Features

- Modern, minimal UI for a premium look
- Fully responsive design (desktop, tablet, mobile)
- Property listing grid with reusable cards
- Search/filter interface (UI-based)
- Services section explaining business offerings
- Statistics section for credibility
- Testimonials for social proof
- Inquiry form for lead generation
- Strong call-to-action sections
- Clean component-based architecture

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
